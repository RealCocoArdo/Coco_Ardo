"Bruteforce" Bitcoin private key

This is a small fun project of mine.

How it works:
  • It takes a random input. For example common names or passwords
  • Then it uses sha256 to create a private key of it (how it was done in the old days)
  • It creates a Bitcoin adress of that private key
  • It checks on mempool.space if there is or was money on that adress
  • It stores a .txt file of Succesfull and allready looted private keys
